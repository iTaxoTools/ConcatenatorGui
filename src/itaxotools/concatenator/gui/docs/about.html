Perform a sequence of transformations on a series of input files:

• Convert between different file formats

• Align sequences per character set

• Rename or delete character sets

• Subset character sets by codon

This GUI is a development prototype and is not yet connected to the Concatenator core functionalities.

• Random character sets will be generated for each file and progress step.
• No changes will be written to disk.
